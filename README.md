# google-map-api-postcards

## to do: 
- overview of what APIs are 
- overview of Google Maps API (+ places API)
- coding portion; building out a website that 
    - connects to the Google Maps API to embed a basic map in the site (for sample city)
    - connects to the places API to pull some pictures from google maps for the postcards (for sample city)
    - takes user input for any city 
    - create custom borders with CSS for postcards 
    - create a caption of the city picture is from 
    - create caption of what the picture is of in the city (Eiffel Tower in addition to Paris, France)
    

## stretch: 
- be able to send the postcard via email + print out 
- be able to add in mail formatting for the back of a printed out postcard (to + from addresses)
- be able to add in custom message before printing them
